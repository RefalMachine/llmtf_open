Мы предоставляем функционал автоматической оптимизации промптов.

# Базовое использование

Пример использования вы можете найти в файле `example.py`.

# Основные компоненты

- `optimizer.py` — основной класс для оптимизации
- `optimizable_task` — абстрактные классы задач
- `solution_database.py` — реализация базы данных решений
- `sampler.py` — класс для сэмплирования решений
- `runner.py` — класс для генерации ответов моделей
- `evaluator.py` — класс для оценки ответов моделей и генерации отзывов
- `mutate.py` — класс для мутации решений

- `view_evolution.py` — скрипт для сбора логов эволюции в таблицу/график

# Поддерживаемые методы генерации

На данный момент фреймворк поддерживает только задачи на генерацию текста (`generate`).

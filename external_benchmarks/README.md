# External benchmarks

В этой директории находятся внешние бенчмарки, подключенные как git submodule.

## RuWikiBench
Русский бенчмарк для оценки аналитических способностей больших языковых моделей через генерацию статей в стиле Википедии.

Основные этапы оценки:
- ранжирование релевантных источников для статьи;
- построение связного плана статьи;
- генерация текста секций на основе предоставленных источников.

Метрики:
- NDCG, R-Precision
- BERTScore, ROUGE-L, BLEU

Внутри репозитория есть:
- инструкция по установке данных бенчмарка и запуска оценки в README;
- скрипты для запуска через консоль/jupyter notebook
- демонстрация запуска

### Как запускать

1. Перейдите в сабмодуль:

   ```bash
   cd external_benchmarks/ruwikibench
   ```

2. При первом запуске подготовьте окружение:
   - скачайте файл с данными бенчмарка;
   - разложите JSON в рабочую структуру, используя предоставленные скрипты;
   - подготовьте индексы, сниппеты и эмбеддинги с помощью prepare_env().

3. После подготовки окружения запуск можно делать:
   - через `main.ipynb`;
   - через `local_benchmark_vllm.ipynb` (локальный запуск через vLLM);
   - через CLI-скрипт.

4. Используется основной класс `WikiBench`.

## RuBookSum
Бенчмарк для автоматического реферирования художественной литературы с помощью LLM.

Что сравнивается:
- иерархический метод;
- blueprint ("чертёжный" метод);
- иерархический метод с фильтрацией близких узлов;
- blueprint с кластеризацией вопросов.

Основные метрики:
- ROUGE-L, BERTScore.

Внутри репозитория есть:
- инструкция по установке датасета книг и рефератов (`combined_data.json`) и запуска оценки в README;
- скрипты для запуска через консоль/jupyter notebook
- демонстрация запуска

### Как запускать

1. Перейдите в сабмодуль:
   ```bash
   cd external_benchmarks/rubooksum
   ```

2. Убедитесь, что датасет `combined_data.json` находится в рабочей директории.

3. Запуск можно делать:
   - через `main.ipynb`;
   - через `local_benchmark_vllm.ipynb` (локальный запуск через vLLM);
   - через CLI-скрипт.

4. Используется основной класс Summarisation
